========================================================================
                      MarkItDown Document Converter
========================================================================

Thank you for downloading MarkItDown Document Converter!

This folder contains a standalone Windows GUI application to convert
various document types to Markdown format.

------------------------------------------------------------------------
WHAT'S IN THIS BUNDLE?
------------------------------------------------------------------------
* markitdown.exe         - The main desktop converter application.
* pandoc.exe             - The Pandoc conversion engine (pre-configured).
* LICENSE-THIRD-PARTY.md - Licensing notices for all components.
* README.txt             - This instruction file.

------------------------------------------------------------------------
HOW TO INSTALL
------------------------------------------------------------------------
No installation is required! This application is fully portable.
1. Extract the contents of this ZIP archive to a folder of your choice
   (e.g., C:\Program Files\MarkItDown or your Desktop).
2. Keep the "pandoc.exe" file in the same folder as "markitdown.exe".
   This allows the converter to automatically detect it at startup.

------------------------------------------------------------------------
HOW TO USE (GUI)
------------------------------------------------------------------------
1. Double-click "markitdown.exe" to open the GUI.
2. Add files to the queue:
   - Click "Ajouter Fichier(s)" to select files.
   - Click "Ajouter Dossier" to select a whole folder recursively.
   - OR simply drag-and-drop files/folders directly from Windows 
     Explorer into the application window!
3. Select your output folder by clicking "Dossier de sortie".
4. Click "Convertir" to run the conversion.
5. In the final result popup, double-click any file path to open the
   converted Markdown file immediately in your default editor.

------------------------------------------------------------------------
SUPPORTED FORMATS
------------------------------------------------------------------------
* Native Support (works anywhere):
  - PDF (.pdf)
  - Word (.docx)
  - Excel (.xlsx)
  - HTML (.html, .htm)
  - Text (.txt, .md)

* Enhanced Support (powered by the included pandoc.exe):
  - PowerPoint (.pptx)
  - Rich Text (.rtf)
  - EPUB Books (.epub)
  - OpenDocument Text (.odt)
  - LaTeX (.tex)
  - MediaWiki (.wiki)
